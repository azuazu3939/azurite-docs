# 仮想ロア・カスタムエンチャント

装備やアイテムの価値を見やすく伝える要素です。  
仮想ロアによる補助表示と独自エンチャントによって、プレイヤーが装備を判断しやすくします。

## この要素が担うこと
- 採集道具や鍛造品の情報をロアとして分かりやすく重ねる
- 独自エンチャントで装備ビルドの個性を増やす

## プレイヤーから見る流れ
- アイテムを拾う、作る、強化する
- ロア表示を見て使い道や価値を判断する
- 必要ならエンチャントを組み合わせて役割を尖らせる

## つながる要素
- 経済系の価値付けとは独立して、装備情報の見え方を整える
- 採集や鍛造の結果はこの要素を通して見え方が整理される
- 個別のロア行やエンチャント定義は [設定項目](./reference.md) を参照
